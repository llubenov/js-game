How to play

The game is played by moving the players' rectangle by using - WASD keyboard buttons.


About

This game has been made as part of my university coursework for Client Side Graphics Programming for the Web Module. Main task was to create an interactive game using HTML5 and JavaScript. The idea of the game is to collect as many circles (points) as possible before it starts getting really difficult to collect and the gravity starts increasing, which will make the player to start falling down and this will make it quite hard for the player to be collecting the circles.
